API Documentation

Use the URL below along with http://localhost:5009 to get the appropriate dataset. The Response generates is a JSON format.

Route URL Description
Precipitation /api/v1.0/precipitation Returns Precipitation Value for last 12 months grouped by date.
To get the Precipitation data, use the below url -
/api/v1.0/precipitation
Stations /api/v1.0/stations Returns a JSON list of stations from the dataset.
To get the Stations data, use the below url -
/api/v1.0/stations
Temperature /api/v1.0/tobs Query for the dates and temperature observations from a year from the last data point. Returns a JSON list of Temperature Observations (tobs) for the previous year.
To get the Temperature data, use the below url -
/api/v1.0/tobs
Temperature Calculations /api/v1.0/Start Date
/api/v1.0/Start Date/End Date		 Return a JSON list of the minimum temperature, the average temperature, and the max temperature for a given start or start-end range. The date should be in Year-Month-Day (YYYY-MM-DD) format.

/api/v1.0/Start Date
When given the start only, calculate TMIN, TAVG, and TMAX for all dates greater than and equal to the start date.
For Example -
/api/v1.0/2016-12-25
Returns the temperature min,ave and max for the time period greater than, or equal to 2016-12-25.

/api/v1.0/Start Date/End Date
When given the start and the end date, calculate the TMIN, TAVG, and TMAX for dates between the start and end date inclusive.
For Example -
/api/v1.0/2016-12-25/2016-12-30
Returns the temperature min,ave and max for the time period between 2016-12-25 and 2016-12-30, including the two dates.
